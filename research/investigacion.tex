\begin{abstract}
% El resumen ahora con Sans Serif y más aire
{\sffamily\fontsize{9.5pt}{12pt}\selectfont
Este documento técnico presenta la implementación del ecosistema \textbf{DomoNode}. Se detalla el flujo de trabajo bajo \textbf{Fedora Linux}, el uso de compiladores \textbf{XC8} y la integración de servicios en la nube como \textbf{Koyeb} y \textbf{Supabase} para evitar el \textit{cold start}. El objetivo es demostrar que la documentación técnica de alta fidelidad y bajo peso (34.4 kB) es un estándar de calidad en la ingeniería moderna.}
\end{abstract}

\section{\sffamily Introducción}
\IEEEPARSTART{E}{l} desarrollo de sistemas domóticos requiere una simbiosis entre el hardware de bajo nivel y el software de gestión. DomoNode utiliza microcontroladores de 8 bits para tareas atómicas, garantizando eficiencia energética y robustez.
